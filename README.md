# Deviens Dev

Built with [Gatsby](https://www.gatsbyjs.org/).

## Getting Started

- `yarn`: install dependencies.
- `yarn start`: launch Gatsby in `development` mode.

:bulb: You can use `http://localhost:8000/___graphql` for debug the GraphQL queries
